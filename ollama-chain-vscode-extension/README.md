## Ollama Chain VS Code Extension

This extension integrates Ollama Chain with VS Code to enhance development workflows. It provides tools for interacting with Ollama models directly within the editor.

### Installation

1. **Build the extension**: Run `npm run package` to generate a `.vsix` file in the `dist` directory.
2. **Install manually**: Open VS Code, go to the Extensions view (`Ctrl+Shift+X`), click the three dots (`...`) and select **Install from VSIX...**. Choose the generated `.vsix` file to install.

### Usage

Once installed, trigger the extension's command by typing `Ctrl+Shift+P` and selecting **Hello World** from the command palette.
